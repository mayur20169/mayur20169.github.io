Machine Learning

Machine learning is a branch of artificial intelligence (AI) that focuses on the development of computer programs that can access data and use it to learn for themselves. The process of machine learning involves feeding vast amounts of data into algorithms and then allowing them to learn and make decisions independently. This technology is used in a variety of applications, from self-driving cars to face recognition and natural language processing.

Deep Learning

Deep learning is a subset of machine learning that uses multiple layers of artificial neural networks to progressively extract higher-level features from raw input data. It is a type of artificial intelligence (AI) which enables computers to learn from large amounts of data without relying on task-specific rules. Deep learning algorithms are capable of learning from data that is unstructured or unlabeled and can produce better results than traditional machine learning algorithms.

Computer Vision

Computer vision is a field of artificial intelligence that deals with the analysis of digital images and videos to gain an understanding of the objects within them. It applies techniques from machine learning, image processing, and deep learning to recognize objects, detect faces, and classify scenes. Computer vision is used in a variety of applications, such as self-driving cars, robotics, and medical imaging.

Natural Language Processing

Natural language processing (NLP) is a field of artificial intelligence that deals with the interaction between computers and human (natural) languages. It is used to analyze, understand, and generate human language in a smart and useful way. NLP tasks include automatic summarization, part-of-speech tagging, parsing, entity recognition, machine translation, sentiment analysis, and question-answering.

Project 2

SemEval2023 Task9: Multilingual Tweet Intimacy Analysis

The goal of this task is to predict the intimacy of tweets in 10 languages. We are given a set of tweets in six languages (English, Spanish, Italian, Portuguese, French, and Chinese) annotated with intimacy scores ranging from 1-5 to train your model.} The model performance will be evaluated on the test set in the given 6 languages as well as an external test set with 4 languages, not in the training data (Hindi, Arabic, Dutch, and Korean).} We will use Pearson's r as the evaluation metric.
